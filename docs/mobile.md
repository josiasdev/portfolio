# Mobile Documentation

Este documento descreve as estratégias e padrões utilizados para garantir a melhor experiência mobile (Responsividade).

## Estratégias
- Tailwind `md:`, `lg:`, `sm:` utilities para Breakpoints
- Navegação convertida para Side Menu (Sheet) em telas menores
- Cursor Invertido (CustomCursor) desativado para devices Touch (`pointer: fine`)
